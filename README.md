# Portfolio V3 (sans JavaScript)

Cette version n'utilise aucun JavaScript, donc tout s'affiche même en double-clic (file://).

## Modifier rapidement
- Texte principal : `index.html` (CTRL+F pour trouver les sections)
- Images : `assets/images/` (remplace les fichiers en gardant le même nom)
- CV : `assets/resume/`

## Option contenu
Le fichier `data/content.json` est fourni (pour référence). Si tu veux, je peux régénérer automatiquement le HTML à partir de ce JSON quand tu auras fait des changements.
